Behold! A Key Sequence Detector

Simply type the key sequence, 'easter egg', to trigger the action passed to the plugin. Anywhere!

Code: $(window).keySequenceDetector('easter egg', function(){ alert('boom'); });

Scoped to an element

Now the key phrase is 'easter eggs', but it must be entered in this input:

(Note: the first, window-bound detector will "boom" first, then this one will "boom" again when you enter the final "s"

Code: $('input').keySequenceDetector('easter eggs', function() { alert('booms'); });
